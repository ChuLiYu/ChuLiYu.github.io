Markdown 試寫與筆記

來測試一下各種 Markdown 語法

  • 標題1~6: # text (1個＃到6個#)

    Heading1

    Heading2

    Heading3

    Heading4

    Heading5
    Heading6

  • 代碼: `text`

    text

  • 引用： >text

    text

  • 粗體： **text**

    text

  • 連結： [text](https://ChuLiYu.github.io)

    text

  • 斜體： *text*

    text

  • 刪除： ~~text~~

    text

  • 項目符號： - text

    • text

  • 項目編號： 1. text
